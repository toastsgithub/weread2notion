# 将微信读书划线和笔记同步到Notion


本项目通过Github Action每天定时同步微信读书划线到Notion。

预览效果：https://book.malinkang.com

> [!WARNING]  
> 请不要在Page里面添加自己的笔记，有新的笔记的时候会删除原笔记重新添加。


## 使用 (如何更新 Cookie)

> [!IMPORTANT]
> 更新 WEREAD_COOKIE: 访问 weread, 等它任意发一个请求，查看 req 里携带的 cookie, 整个 Copy 到 secret:WEREAD_COOKIE 里
